# go-stocks

A simple program for stock quotes using mfinance.com.br API (https://mfinance.com.br/swagger/index.html)

## Installation

- Install *go 1.15* from https://golang.org/
- Clone this repository
- cd into the folder
- Build and run:
```
go build
./go-stocks
```

## Usage

```
./go-stocks petr4 azul4

     PETR4:    19.75
     AZUL4:    25.73
```
